# backend
---------

### Setup

1. Clone the repository
2. Disable line ending normalization
```
git config core.autocrlf false
```
3. Open the project in Intellij IDEA

### Working with the project

Start app:
```
./gradlew run
```
or run the main class from IDE.

Run tests:
```
./gradlew test
```
or run tests from IDE.

Build far jar:
```
./gradlew shadowJar
```
